# VN Realm

A lightning-fast visual novel search and discovery platform powered by the VNDB API.

## Overview

VN Realm helps you search for visual novels and discover detailed information including synopsis, ratings, releases, and more. The site also provides torrent links for easy access to visual novels.

## Features

- **Fast Search**: Search through thousands of visual novels from the VNDB database
- **Detailed Information**: View comprehensive details including ratings, tags, platforms, languages, and more
- **Release Information**: Browse official and unofficial releases with platform and language details
- **Torrent Integration**: Find torrent links directly from search results
- **Clean UI**: Minimal and modern interface for easy navigation

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **VNDB API** - Visual novel database

## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (or npm/yarn)

### Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## API Credits

This project uses the [VNDB API](https://vndb.org/d11) for visual novel data. VNDB is a comprehensive database for visual novels maintained by the community.
